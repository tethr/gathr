.. _glossary:

Glossary
========

.. glossary::
   :sorted:

   datastream
     A collection of fields.  Represents a stream of data collection where each
     row is a particular set of data points collected at a single time and each
     column is a field.

   field
     A single datapoint collected by `Gathr`.

   form
     A screen where data is entered.

   metadata
     Description of data to be collected and the protocol used to collect it, 
     required by `Gathr`. Defined in YAML files.

   resource
     A thing about which data is collected.  Resources are composed into a 
     hierarchical tree structure.
